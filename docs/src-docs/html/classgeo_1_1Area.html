Detailed Description

An Area object is a set of rectangular bounds used to check for geospatial point inclusion. Areas are constructed from line strings, where each segment has it's own rectangular linear ring. Areas are invalid if the line string has less than 2 points, the width is less than or equal to zero, or the extension is less than zero.
