Detailed Description

A KML file.

KML Color Values are specified as follows: color_value is easiest to express in hex: 0xaabbggrr where aa -> is the transparency alpha. 00 is transparent; ff is fully opaque. where bb -> BLUE where gg -> GREEN where rr -> RED This is backwards from how it is usually specified, i.e., RGB.
